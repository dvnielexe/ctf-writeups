# 🧠 CTF Writeups Repository

A collection of Capture The Flag (CTF) writeups and solutions from various platforms.
This repo documents my approach to solving challenges, the tools used, and the lessons learned along the way.

---

## 📌 Platforms Covered

* OverTheWire
* picoCTF
* TryHackMe
* Hack The Box
* Other CTF events and platforms

---

## 🎯 Purpose

* Track progress and consistency in CTF practice
* Document thought processes and problem-solving approaches
* Build a public knowledge base for future reference
* Help others understand how different challenges can be approached
