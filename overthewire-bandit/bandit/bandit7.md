# Level 7 → Level 8

## Objective
Find the password stored in `data.txt` next to the word `millionth`.

## Key Concept
`grep` is used to search for specific patterns or words inside files.

## Command Used

```bash
grep "millionth" data.txt
```

## Explanation
`grep` searches through the file and returns the line containing the word `millionth`.

The password appeared on the same line, making it easy to identify.

## What I Learned
- `grep` is useful for quickly filtering text
- Pattern searching is an important Linux skill
- Large files become easier to analyze using filtering tools
